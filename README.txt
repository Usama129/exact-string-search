Scripts aimed at analysing runtime differences between exact string search algorithms
- Brute-force
- Knuth-Morris-Pratt
- Rabin-Karp

Make for Windows: run 'make' in this directory to run the script

To run without using the makefile, use 'python hw2.py -i T.fa -p P.fa'

For help, run 'python hw2.py -h'
